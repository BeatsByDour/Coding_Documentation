## Component / Namespace

- Component: GNU coreutils
- Namespace: Bash shell

## Beschrijving

Maakt een leeg bestand aan of update de timestamp van een bestaand bestand.

## Signatuur

```
touch bestand
```

## Parameters

- **bestand** – string – Bestandsnaam.

## Retourwaarde

- Creëert bestand of wijzigt metadata.

## Gedrag / Opmerkingen

- Veel gebruikt om snel nieuwe scripts of configfiles te maken.

## Voorbeeld

```
touch notes.md
```
